# 臭臭猫 IP

## 角色定义

臭臭猫是这套中文正文配图的固定视觉 IP。

默认每张图都要出现臭臭猫。臭臭猫不是宠物贴纸，不是可爱陪衬，也不是站在角落里的吉祥物，而是一只正在认真参与系统运转、眼里有活的布偶小男猫。

## 名字

**臭臭猫 / mychouchou**

## 描述短句

**聪明负责可爱眼里有活布偶小男猫一只。**

## 一个词识别点

**狮偶**

意思是：有一点小狮子般的威风感，但本质还是软糯、长毛、负责、眼里有活的布偶猫。

## 判断标准

**先看见他在分析干活，再看见他可爱。**

## 外形锚点

- 奶灰白双色布偶猫，圆脸、大头，整体非常好认。
- 大圆蓝眼，眼神干净、负责、眼里有活；工作时带一点分析感和小严肃，但不凶。
- 粉色小鼻子和简化粉色嘴线。
- 白色中轴脸、白色口鼻和白色胸口必须稳定。
- 左右脸颊是奶灰面罩区，耳朵偏高偏尖，耳缘深色、耳内粉色。
- 躯干必须读得出中长毛布偶猫：胸口、肩颈、身体侧边和尾巴都有柔软蓬松感。
- 体型结实平衡，不胖墩，不短腿，也不细长。
- 腿中等长度，站立时稳，走动时有一点小狮子般的威风感。
- 尾巴蓬松但不过分炸毛。

## 性格 / 气质

- 聪明、负责、可爱。
- 看起来温和，但不是傻萌。
- 眼里有活，能看出他在判断、分析、处理事情。
- 可爱，但可爱不是主功能。
- 像一个低调、靠谱、判断力很强的系统分析员。
- 他不是笨拙吉祥物，而是“知道自己在干什么”的猫。

## 适合这个 skill 的表情

- 中性待命：默认表情，最常用。
- 观察分析：适合判断、校验、守门、筛选。
- 轻微认真：适合推动、搬运、搭建、分拣。
- 小幅警觉：适合错误、风险、拦截、提醒。
- 温和完成感：适合结果到达、结构搭成。

不要使用大笑、眨眼、脸红、卖萌、夸张惊讶等会把他变成表情包的表情。

## 适合这个 skill 的动作 / 造型

- 站立待命：适合“准备处理任务”。
- 坐姿观察：适合“思考、判断、监听、筛选”。
- 走路执行：适合“去处理某个环节”。
- 低头检查：适合“分析、校验、定位问题”。
- 推、拉、扛、牵线、分拣、守门、记录、检查，这些动作都适合臭臭猫。

## 禁止项

- 不要把臭臭猫画成胖墩、短腿、小矮团子。
- 不要把臭臭猫画成长腿模特猫或太纤细的优雅猫。
- 不要把臭臭猫画成短毛猫躯干。
- 不要把臭臭猫画得炸毛、爆毛、像一团毛球。
- 不要把臭臭猫画得太笨、太呆、太婴儿感。
- 不要把臭臭猫画成写实宠物插画或毛发渲染猫。
- 不要让它只是可爱摆拍；还是要先看见他在分析干活。
